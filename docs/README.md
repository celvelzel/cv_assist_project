## Writing Convention

When adding a new report:

1. Use date-first naming: `YYYY-MM-DD_topic.md`.
2. Include metadata at the top: Date, Author, Topic.
3. Keep one report focused on one milestone/theme.
4. Link important reports from the root `README.md` when needed.
